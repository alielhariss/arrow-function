## ✨ يعني إيه Arrow Function؟  
- **Arrow Function (دالة مختصرة):** طريقة نكتب بيها الدوال (methods أو functions) في سطر واحد باستخدام `=>`.  
- بتوفر وقت ومساحة وبتخلي الكود أبسط وأسهل للقراءة.
